The Common LaTeX Service Interface is a web service providing access to a LaTeX compiler.
This CLSI server runs on the OpenShift network as a free public service.

How to get access

To use the CLSI you will need a developer token. To request a token, please email me at bardellie@gmail.com.

How to report a missing packet

At this time, this CLSI runs a striped-down version of LaTeX. To request the installation of a missin packet, please email me at bardellie@gmail.com.

Client

If you are looking for a client to access this CLSI server, check out my CLSI client on GitHub.
